# 바이브 코딩 (Vibe Coding)

![Vibe Coding Hero](https://github.com/hslcrb/whatisvibecoding/raw/main/public/vibe_coding_hero.png)

> **"코딩은 이제 타이핑이 아니라 '바이브'입니다."**

본 프로젝트는 AI와 인간의 교감이 만들어내는 새로운 산업 생산 패러다임, **바이브 코딩(Vibe Coding)**을 소개하고 그 경제적 당위성과 미래 지형도를 제시하는 몰입형 랜딩 페이지입니다.

## 주요 특징

- **압도적 비주얼**: WebGL(Three.js) 기반의 실시간 유체 배경과 3D 인터랙티브 요소.
- **최신 지능 지표**: 2026년 2월 기준 최신 AI 모델(Claude 4.6, Codex 5.3, Gemini 3.1)의 실전 벤치마크 데이터 수록.
- **몰입형 서사**: 단순한 기술 나열을 넘어, AI 시대에 인간 지휘자가 갖춰야 할 '직관(Vibe)'의 가치를 설득력 있게 전달합니다.
- **에이투지체(A2z) 시스템**: 현대적 미학의 정수인 에이투지체를 전역에 적용하여 극강의 타이포그래피 경험을 제공합니다.

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- **Framework**: Next.js (App Router)
- **Styling**: Vanilla CSS (Global & Modules)
- **Interactivity**: Three.js (WebGL), Intersection Observer API
- **Typography**: A2z Custom Font Family
