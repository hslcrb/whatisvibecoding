# 바이브 코딩 (Vibe Coding)

![Vibe Coding Platform](https://github.com/hslcrb/whatisvibecoding/raw/main/public/website.webp)

> **"지능의 중력을 거스르고, 가치의 바다로 항해하십시오."**

본 프로젝트는 AI와 인간의 교감이 만들어내는 새로운 문명적 패러다임, **바이브 코딩(Vibe Coding)**을 선포하고 그 경제적 필연성과 미래 지형도를 제시하는 몰입형 디지털 선언문입니다.

## 🌌 Core Vision

- **Overwhelming Beauty**: WebGL(Three.js) 기반의 실시간 유체 동역학 배경과 고품질 인터랙티브 레이아웃.
- **Intelligent Benchmarks**: 2026년 2월 기준 최신 초지능 모델(Claude 4.6, Codex 5.3, Gemini 3.1)의 실전 코딩 벤치마크 데이터 수록.
- **Narrative Excellence**: 단순한 기술 나열을 넘어, AI 시대의 '지휘자'가 갖춰야 할 직관(Vibe)의 가치를 심도 있는 서사로 전달합니다.
- **A2z Typography**: 고품질 타이포그래피 에이투지체(A2z) 시스템을 전역에 적용하여 정제된 가독성을 제공합니다.

## 🚀 Get Started

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 🛠 Tech Stack

- **Framework**: Next.js (App Router)
- **Visuals**: Three.js (WebGL Fluid Engine), Framer Motion-inspired CSS
- **Interactions**: Intersection Observer API, Fluid Scaling (clamp)
- **Identity**: A2z Custom Font System, Rheehose Design Language
